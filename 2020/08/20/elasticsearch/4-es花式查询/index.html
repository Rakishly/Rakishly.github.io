3、分页查询

数据下标还是从0开始，跟数据库一样

  • from 从第几个开始
  • size 返回多少条数据 相当于数据库的 limit 0, 2

4、布尔值查询

must （and），所有的条件都要符合 where id = 1 and name = xxx

多条件查询

should （or），所有的条件都要符合 where id = 1 or name = xxx

must_not 也就是 not 查询

查询 age 不是 18 的

过滤器 filter，对数据进行过滤

获取 name = rakishly 的数据，过滤出 age >= 20 and age <= 25

  • gt 大于
  • gte 大于等于
  • lt 小于
  • lte 小于等于！

5 、匹配多个条件

使用空格隔开，只要满足其中一个结果既可以被查出，可以通过分值基本的判断

6、精准查询

term 查询是直接通过倒排索引指定的词条进程精确查找的！

关于分词：

  • term 直接查询精确的
  • match 会使用分词器解析！（先分析文档，然后在通过分析的文档进行查询！）

使用 match 查询就会进行分词，比如”php是世界上最好的语言”会被分词为”php 是 世界 上 最好 的 语言”, 所有有关”php 是 世界 上 最好 的 语言”, 那么所有包含这几个词中的一个或多个的文档就会被搜索出来。

term 查询 会去倒排索引中寻找确切的term，它并不知道分词器的存在，这种查询适合keyword、numeric、date等明确值。
